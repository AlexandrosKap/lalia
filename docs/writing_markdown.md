# Writing Markdown for Lalia

## Line Types

### Text Line

```md
Hello hello!
```

### Label Line

```md
# LABEL
```

### Menu Line

```md
* LABEL1|Choice number 1.
* LABEL2|Choice number 2.
* LABEL3|Choice number 3.
```

### Comment Line

```md
[](My comment.)
```

### Jump Line

```md
[](#LABEL)
```

### Variable Line

```md
![var](myVar|1)
![var](1)
```

### Calculation Line

```md
![calc](myVar|1 + 2)
![calc](1 + 2)
```

### Procedure Line

```md
![proc](myVar|foo 1 2 3)
![proc](foo 1 2 3)
```

### Check Line

```md
![check](4 = 4)
```

### Pause Line

```md
***
```
